    R에서 rjson package 의 toJSON(), fromJSON() 함수 알아보기

    02 Mar 2016 » R

    JSON 포맷의 데이터를 R에서 읽어 들인 후 가공하기 위해 JSON 형식을 parser 해야 하는 과정이 필요하다고 한다.
    rjson 라는 패키지가 이를 도와주는데 이 안에 있는 함수 toJSON(), fromJSON() 2개를 기본적으로 이용하여 실습을 해보겠다.

    A) 준비

    A-1) 예제 데이터

    실습을 하기 위하여 아래와 같은 JSON 타입의 데이터가 있다고 가정해 보겠다.
    아래의 데이터는 JSON 을 설명하는 위키백과의 예제데이터를 적절히 수정한 것이다.

    [
  {
    "Name": "Test",
    "Age": 25,
    "Sex": "F",
    "Address": "Seoul",
    "Hobby": "Basketball"
  }
]

    이것을 워킹디렉토리에 test.json 로 저장하고 이를 R에서 적절하게 사용하기 위해 불러와 보겠다.
    어떠한 텍스트에디터를 이용하던 상관없다, 저장방법은 위의 JSON 형식 텍스트를 복사한 다음 "test.json" 이란 파일명으로 저장만 하면 된다.

    나는 Sublime text 3 라는 텍스트 에디터를 이용해 이처럼 저장하였다.

    A-2) jsonlite 패키지 불러오기

    당연히 rjson 패키지를 불러오는 것이 필요하다.

    library("rjson")

    B) toJSON(), fromJSON() 함수 이용해보기

    B-1) fromJSON()

    우선 fromJSON() 함수를 알아보자.
    JSON 형 데이터를 R에서 가공할 수 있도록 형태를 변환하여 준다.

    data <- fromJSON(file="test.json")
data
    ## [[1]]
## [[1]]$Name
## [1] "Test"
## 
## [[1]]$Age
## [1] 25
## 
## [[1]]$Sex
## [1] "F"
## 
## [[1]]$Address
## [1] "Seoul"
## 
## [[1]]$Hobby
## [1] "Basketball"
    class(data)
    ## [1] "list"

    list 형식으로 반환되는데 적절히 data.frame 형태로 변환하여 보면 다음과 같다.

    dataframe <- as.data.frame(data)
dataframe
    ##   Name Age Sex Address      Hobby
## 1 Test  25   F   Seoul Basketball

    B-2) toJSON()

    toJSON() 은 이와 반대의 성격을 가진다.
    R의 객체를 특히 list, data.frame 등을 JSON 형으로 변환시킨다.
    방금만든 dataframe 객체를 다시 JSON 형태로 바꾸기 위해선,

    toJSON(dataframe)
    ## [1] "{\"Name\":\"Test\",\"Age\":25,\"Sex\":\"F\",\"Address\":\"Seoul\",\"Hobby\":\"Basketball\"}"

    로 하면된다.

    rjson package 말고도 간단히 이용할 수 있는 jsonlite package 도 있는데 이 패키지의 toJSON() 함수는 pretty 하게 반환하는 기능을 가지고 있다.

    jsonlite::toJSON(dataframe, pretty=T)
    ## [
##   {
##     "Name": "Test",
##     "Age": 25,
##     "Sex": "F",
##     "Address": "Seoul",
##     "Hobby": "Basketball"
##   }
## ]

    첫 서두에 있는 예제 데이터와 동일하게 출력된 것을 볼 수 있다.
