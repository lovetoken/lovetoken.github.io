    Amazon Web Service 를 이용한 공동 사용 Rstudio Server 환경구축

    25 Mar 2015 » R

    Amazon Web Service (AWS) 를 이용하여 미리 구축된 Rstudio Server AMI 1 를 배포해주는 사이트가 있다.

    http://www.louisaslett.com

    이 AMI를 이용해 직접 여러대의 Rstudio Server 를 손쉽게 구축시킬 수 있다.

    관련된 정보는 아래링크를 통해 참고하여 알게 되었고 튜토리얼 형태로 이용할 수 있는 방법에 대한 영상까지 있다.

    Brief guide about Rstudio Server's Web interface on AWS : https://www.youtube.com/watch?v=NQu3ugUkYTk

    AWS 를 프리티어로 공짜로 이용하던 이전시절 이영상을 보고 그대로 따라하여 Rstudio Server 를 구축 시켜 봤었다.
    등록되는 도메인을 통해 들어간 후

    ID : rstudio
    P.W : rstudio

    로 로그인하여 자유자재로 웹에서 사용할 수 있게 된다.
    물론 초기 로그인시 아이디와 패스워드는 저렇지만 추후에 다른 패스워드로, 그리고 다른 아이디를 만들어 사용할 수도 있겠다.

    실질적으로 사용하고자 한다면 AWS EC2 사용 과금을 고려해야하며,
    AWS 프리티어 계정기준 t2.micro 와 같은 최저 사양으로 인스턴스를 활성화 시킨다면
    성능, 속도측면에서 기대하지 않는것이 좋을것이다.

    하지만 간편하게 원격으로 R 을 실행할 수도 있고, 소셜코딩이 가능할 수도 있겠으며 간편한 서버증설 등등의
    유용하게 사용할 수 있는 방법은 무궁무진 할 것으로 생각된다.

    1. Amazon Machine Image 의 약어로 어떤 소프트웨어나 환경설정을 이미지화 하여 그대로 재활용할 수 있는 템플릿이다.↩
